ANLEITUNG FÜR HERO-ANIMATION BILDER
====================================

So funktioniert das System:

1. Jeder Ordner repräsentiert einen Event-Typ
2. Der Ordnername = Der Text auf der Rückseite der Karte
3. Legen Sie EIN Bild pro Ordner ab (nur das erste wird verwendet)

BEISPIEL:
---------
Hochzeit/
  └── bild.jpg        ← Wird als "Hochzeit" angezeigt

Geburtstag/
  └── party.jpg       ← Wird als "Geburtstag" angezeigt

Firmenfeier/
  └── event.jpg       ← Wird als "Firmenfeier" angezeigt


WICHTIG:
--------
- Maximale Anzahl: 20 Bilder (= 20 Ordner mit Bildern)
- Aktuell werden die ersten 16 Ordner verwendet
- Bildformate: .jpg, .jpeg, .png, .gif, .webp
- Ordnernamen ohne Umlaute verwenden (ä→ae, ö→oe, ü→ue)

ORDNER UMBENENNEN:
------------------
Wenn Sie den Text ändern möchten, benennen Sie einfach den Ordner um!
Beispiel: "Hochzeit" → "Wedding" = Text wird zu "Wedding"

NEUEN EVENT-TYP HINZUFÜGEN:
---------------------------
1. Neuen Ordner erstellen mit dem gewünschten Text als Namen
2. Ein Bild in den Ordner legen
3. Fertig!
